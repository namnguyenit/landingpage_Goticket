
  # Project Analysis Overview

  This project was migrated from Vite to Next.js (App Router) so it can run and deploy directly on Vercel.

  The original design source is available at:
  https://www.figma.com/design/ehNxR42pnq7rEejJW8lhrM/Project-Analysis-Overview

  ## Local development

  1. Install dependencies:
    `npm install`
  2. Start development server:
    `npm run dev`
  3. Open:
    `http://localhost:3000`

  ## Production build

  1. Build:
    `npm run build`
  2. Run production server locally:
    `npm run start`

  ## Deploy to Vercel

  1. Push the repository to GitHub/GitLab/Bitbucket.
  2. Import the repository in Vercel.
  3. Framework preset: `Next.js` (auto-detected).
  4. Build command: `next build` (default).
  5. Output: `.next` (default).

  No custom Vercel configuration is required for this project.
